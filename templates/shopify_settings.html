Beginner Guide

Connect Shopify step by step

  1. Open your Shopify admin dashboard.
  2. Go to Settings.
  3. Go to Apps and sales channels.
  4. Click Develop apps. If Develop apps is disabled, enable custom app development.
  5. Create a new custom app called BusinessBuilder AI.
  6. Go to Configure Admin API scopes.
  7. Enable the permissions the features you plan to use need:
    • read_products and write_products
    • read_content and write_content
    • read_themes if theme reading is supported
    • write_themes only for supported, approved theme edits
    • read_files if needed
    • write_files only if needed

    Theme safety: Only enable theme write permissions if you understand that theme changes can affect your store design. BusinessBuilder AI should ask for approval before applying theme changes.

  8. Install the app.
  9. Copy the Admin API access token.
  10. Paste the Admin API access token into BusinessBuilder AI in the secure field above.
  11. Enter your Shopify store domain. Example: your-store-name.myshopify.com
  12. Click Test Connection.
  13. After a successful test, use BusinessBuilder AI to create safe draft Shopify assets. Review and approve drafts before supported changes are applied.
Keep your token private. Never share your Shopify Admin API token publicly. Treat it like a password. BusinessBuilder AI stores it securely and never displays it back to you.
How do I find my Shopify token?

In Shopify admin, open Settings → Apps and sales channels → Develop apps, select BusinessBuilder AI, configure the Admin API scopes, install the app, then reveal and copy the Admin API access token. Do not use a Storefront API token.

Troubleshooting a failed connection
  • Check that the store domain uses the format your-store-name.myshopify.com.
  • Make sure the token is the Admin API access token, not a storefront token.
  • Make sure the required scopes are enabled.
  • Make sure the custom app is installed.
  • Make sure your Shopify store is active.
  • If you changed scopes, reinstall the app or update the app token if needed.

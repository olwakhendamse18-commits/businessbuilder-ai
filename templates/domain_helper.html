Quick Reference

DNS terms in plain language

DomainYour human-readable web address.
DNSThe system that points your domain to the correct service.
A recordPoints a name to an IPv4 address.
CNAMEPoints one hostname, often www, to another hostname.
NameserversChoose which provider manages your DNS records.
SSL certificateEnables secure HTTPS connections.
Root redirectSends example.com visitors to www.example.com when needed.
PropagationThe delay while DNS changes update across the internet.
